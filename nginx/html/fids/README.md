# FIDS - Flight Information Display System

Enterprise airport-style flight information system with 4 boards:

- 🛫 **International Departures**
- 🛫 **Domestic Departures**  
- 🛬 **International Arrivals**
- 🛬 **Domestic Arrivals**

## Features
- Real-time auto-refresh every 5 seconds
- Admin panel to add/edit/delete flights
- Modern dark theme with sage green accents
- Responsive design for TVs and monitors

## Quick Start
1. Copy `nginx/` folder to `C:\nginx\`
2. Copy `fids-backend/` folder to `C:\fids\`
3. Run `setup.bat`
4. Open http://localhost

## Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript
- **Web Server:** NGINX (Windows)
- **Backend:** FastAPI (Python)
- **Network:** pfSense, VLAN segmentation
